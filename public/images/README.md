# Images Folder

## Required Images

You need to add these image files to this folder:

1. **neave.jpg** - Your professional headshot
   - Recommended size: 600x600 pixels (square)
   - Format: JPG
   - Should be a professional photo with good lighting

2. **hvac.png** - Screenshot of your HVAC Compliance Dashboard
   - Format: PNG
   - Recommended size: 1200x800 pixels (landscape)

3. **legionella.png** - Screenshot of your Legionella Tracker
   - Format: PNG
   - Recommended size: 1200x800 pixels (landscape)

4. **smoke.png** - Screenshot of your Smoke System Compliance Dashboard
   - Format: PNG
   - Recommended size: 1200x800 pixels (landscape)

## How to Add Images

1. Locate your image files on your Mac
2. Copy them to this folder: `/neave-portfolio/public/images/`
3. Make sure the filenames match exactly (case-sensitive)
4. Refresh your browser to see the images on your website

## Temporary Placeholders

Until you add your real images, the website will show broken image icons. This is normal and expected.
