# Videos Folder

## Required Videos

You need to add these video files to this folder:

1. **hvac.mp4** - Screen recording of your HVAC Compliance Dashboard
   - Format: MP4
   - Duration: 30-60 seconds recommended
   - Should show key features and interactions

2. **legionella.mp4** - Screen recording of your Legionella Tracker
   - Format: MP4
   - Duration: 30-60 seconds recommended

3. **smoke.mp4** - Screen recording of your Smoke System Compliance Dashboard
   - Format: MP4
   - Duration: 30-60 seconds recommended

## How to Record Dashboard Videos

### On Mac (Using QuickTime):

1. Open QuickTime Player (in Applications)
2. Click File → New Screen Recording
3. Click the record button
4. Select the area to record (or entire screen)
5. Open your dashboard and interact with it naturally
6. Click the stop button in the menu bar when done
7. File → Export As → 1080p (or lower for smaller file size)
8. Save as .mp4 format

### Tips for Good Dashboard Videos:

- Keep videos short (30-60 seconds)
- Show the most important features
- Move your mouse slowly and deliberately
- Don't worry about audio (videos will be muted)
- Export in MP4 format for best compatibility

## How to Add Videos

1. Save your screen recordings as MP4 files
2. Rename them to match: hvac.mp4, legionella.mp4, smoke.mp4
3. Copy them to this folder: `/neave-portfolio/public/videos/`
4. The videos will automatically work in the portfolio modals

## File Size Note

Keep video files under 10MB each if possible for faster website loading.
