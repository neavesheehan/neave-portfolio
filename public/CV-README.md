# CV Folder Instructions

## Required File

Add your CV as a PDF file:

**Neave-Sheehan-CV.pdf**

## How to Add Your CV

1. Save your CV as a PDF file
2. Rename it to exactly: `Neave-Sheehan-CV.pdf`
3. Place it in the `/neave-portfolio/public/` folder (this folder's parent)
4. The download buttons on your website will automatically work

## Where the CV is Used

Your CV download is linked in three places:
1. Hero section "Download CV" button
2. Footer "Download CV" link
3. Contact section "Download CV" button

## Note

Until you add your CV, clicking the download button will show a 404 error. This is expected and will work once you add the file.
